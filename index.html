Arduino Robot Car

The program is written in C. The first part is the setup of the pins of the Arduino microprocessor for sensors and actuators. In the loop section the robot checks 2 things: the PIR sensor detects motion and the proximity sensor checks if the distance is bigger than 20 cm. If both requirements are met, than the motors are turned on and the car starts driving forward.

Excel Capacity Planner

At work I all too often heard the question: when do you expect to complete this task? And maybe even more important: when would you be able to start with this next task? To answer this question I used Excel and started filling in the capacity in hours per week for our team. In another sheet everyone can put in their tasks with a deadline and estimated hours. Another sheet is for holidays. With this input I wanted to sum up the hours per week, and to show per team member how much hours they were booked the next coming weeks. The planning is now used everyday and whereas before we would be eager to say that we could put some more tasks in between the workload, now the workload is visible and we're able to give a more realistic estimation when someone could expect our results.

Spiral Conveyor Configurator

In this project I wanted to create a configurable spiral conveyor. Before starting I mapped all the variables that I wanted to be able to customize. Then I mapped the lower and upper limit per variable and also the step-size (increment). The result is shown: the form will show the user a slider per variable and when adjusted the model will update live. This way a new configuration is made in a matter of seconds.
